While you are waiting to get back online, here some cool facts about PWAs:

Self updates

Service Workers can update themselves automatically, woohoo! However, it is best practice to inform the user about a new available version and give her the possibility of choosing when to apply the update. Typically this is done through a UI element, like a button or banner.

Safety first

Service Workers works only on a secure connections (HTTPS). Localhost is considered also secure and developers like it!

Easier to maintain

There is only one version live! No multiple-versions-headaches anymore...
